# ⏱️ Projeto Cronometre

Um cronómetro simples e funcional desenvolvido para praticar conceitos fundamentais de JavaScript, manipulação do DOM e estilização com CSS.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estrutura da página.
- **CSS3**: Design responsivo e estilização.
- **JavaScript**: Lógica do temporizador (start, pause, reset).
